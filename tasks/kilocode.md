# KiloCode — План подключения

## 1. Подключение как провайдер

**Изменения в `opencode.jsonc`:**
```jsonc
{
  "provider": "kilo",
  "providerOptions": {
    // требуется API-ключ от Kilo
  }
}
```

## 2. Импорт агентов и команд

| Источник | Назначение | Функционал |
|----------|------------|------------|
| `agent/triage.md` | Issue triage | Автоматические метки (windows, perf, nix...) + роутинг |
| `agent/translator.md` | Перевод | Локализация с сохранением технических терминов |
| `command/commit.md` | Git commits | Префиксы: docs:, tui:, core:, ci: |
| `command/changelog.md` | Changelog | Генерация из git-коммитов |
| `glossary/ru.md` + 16 др. | Переводы | Правила для 17 языков |
| `tool/github-triage.ts` | Инструмент | Автоматическое типирование |

## 3. Как работает

```
User: "баг в винде"
    ↓
triage агент
    ↓
Результат: +label: windows +assign: catrielmuller
```
