# ОФД провайдеры

## Первый ОФД
- **Сайт:** 1-ofd.ru
- **API:** api.1-ofd.ru/v2
- **Аутентификация:** API key (header)
- **Методы:** receipt, kkt, shift, stat, fn
- **Документация:** по договору

## Такском
- **Сайт:** taxcom.ru
- **API:** api.taxcom.ru/v1
- **Аутентификация:** login + password
- **Методы:** receipt, shift, kkt, stat, fn
- **Документация:** developer.taxcom.ru

## Яндекс.ОФД
- **Сайт:** ofd.yandex.ru
- **API:** api.ofd.yandex.net/v1
- **Аутентификация:** OAuth2
- **Методы:** receipt, shift, stats, kkt
- **Документация:** yandex.ru/dev/ofd

## Эвотор
- **Сайт:** evotor.ru
- **API:** api.evotor.ru
- **Аутентификация:** Bearer token
- **Методы:** receipt, kkt, shift, product
- **Документация:** api.evotor.ru (открытая)
