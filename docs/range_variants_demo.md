# Range Bar — Демонстрация вариантов

## Формат: Demo/Preview Page

**Назначение:** Показать все варианты Range Bar на одной странице для сравнения.

**URL:** `/range_variants_demo`

**Тестовые данные (ETC #10):**
- Entry: 8.0995
- Current: 8.44
- Low: 7.77
- High: 9.78
- Leverage: 10x
- Days: 72

---

## Варианты (10 вариантов + Current)

---

### Variant 0: Segmented Bar — Ценовой диапазон (БАЗОВЫЙ/ИСХОДНЫЙ)

**Инфо:** Bar разделен на 3 зоны по ценовым уровням: Low→Entry, Entry→Current, Current→High. Подсвечен диапазон Entry→Current (зеленый сегмент выше на 10px + glow).

**Плюсы:** Наглядно показывает положение Entry и Current относительно Low/High
**Минусы:** Не отображает распределение дней по ROE

**Цвета:**
- Серо-красный (#fca5a5) — Low→Entry
- Зеленый (#22c55e) — Entry→Current (подсвечен, 60px)
- Серо-зеленый (#bbf7d0) — Current→High

**Разметка:**
```
┌─────────────────────────────────────────────────────────┐
│ [████ LOW ████][████ ENTRY ████][████ CURRENT ████][░░]│
│  30d/41.7%     E:8.0995      C:8.44          High:9.78 │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 1: Segmented Bar

**Инфо:** Bar разделен на сегменты по ROE-диапазонам. Маркеры ENTRY/CURRENT нанесены вертикальными линиями.

**Плюсы:** Показывает распределение дней по ROE-диапазонам
**Минусы:** Не отображает ценовые уровни

```
┌─────────────────────────────────────────────────────────┐
│ [███ -10→0% ███][███ 0→+10% ███][███ +10→+50% ███]     │
│    12d/16.7%       27d/37.5%        33d/45.8%            │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 2: Vertical Timeline

**Инфо:** Вертикальный список, каждая сделка = Symbol + ROE% + Bar внутри. Похож на Calendar Heatmap.

**Плюсы:** Компактный, показывает несколько уровней
**Минусы:** Занимает вертикальное пространство

```
┌─────────────────────────────────────────────────────────┐
│ [████████████████ ENTRY +0%                          ] │
│ [████████████ CURRENT +42%                            ] │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 3: Mini-Candles Strip

**Инфо:** Bar из 72 мини-сегментов (1 день = 1 сегмент), без отступов. Цвет каждого сегмента по ROE этого дня.

**Плюсы:** Показывает все дни, историю движения
**Минусы:** Может быть "шумным" при 70+ днях

```
┌─────────────────────────────────────────────────────────┐
│ │▌│▌│▌│▌│░│▌│▌│▌│░│▌│▌│▌│░│▌│▌│▌│░│▌│▌│▌│░│▌│▌│▌│     │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 4: Gradient Bar

**Инфо:** Bar с единым градиентом (red→gray→green). Позиция ENTRY/CURRENT показана линиями сверху/снизу.

**Плюсы:** Простой, наглядный
**Минусы:** Размывает границы диапазонов

```
┌─────────────────────────────────────────────────────────┐
│ 7.77 [███████████████████████████████] 9.78            │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 5: Stacked Horizontal (seaborn style)

**Инфо:** Горизонтальные stacked bars. Сегменты = агрегированные дни по ROE-диапазонам. Как matplotlib/seaborn stacked bar.

**Плюсы:** Показывает пропорции диапазонов
**Минусы:** Не видно последовательности дней

```
┌─────────────────────────────────────────────────────────┐
│ ████████████████│████████████████████████│             │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 6: Timeline Strip

**Инфо:** Горизонтальная полоса с маркерами ENTRY и CURRENT как вертикальными линиями. Bar между маркерами показывает диапазон Low→High.

**Плюсы:** Чётко показывает позицию входа и текущую
**Минусы:** Не показывает историю дней

```
┌─────────────────────────────────────────────────────────┐
│  ────────[ENTRY]─────────────────────[CURRENT]───────  │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 7: Filled Range

**Инфо:** Bar залит от Low до High цветом, маркеры ENTRY/CURRENT делят bar на зоны. Цвет меняется от красного к зелёному.

**Плюсы:** Наглядно показывает положение текущей цены
**Минусы:** Не видно распределения дней

```
┌─────────────────────────────────────────────────────────┐
│ 7.77 [███████████████ ENTRY ████████ CURRENT] 9.78    │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 8: Markers Only

**Инфо:** Только вертикальные линии ENTRY и CURRENT на ценовой шкале. Bar фона показывает диапазон Low→High.

**Плюсы:** Максимально простой
**Минусы:** Минимум информации

```
┌─────────────────────────────────────────────────────────┐
│ 7.77 ─────────────────────────────── 9.78              │
│       │ENTRY│              │CURRENT│                  │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 9: Candles Range

**Инфо:** Мини-свечи (high/low/body) как полоски внутри bar. Каждая "свеча" = один день.

**Плюсы:** Показывает high/low каждого дня
**Минусы:** Сложнее для восприятия

```
┌─────────────────────────────────────────────────────────┐
│ │▔▔│▔▔│▔▔│░│▔▔│▔▔│▔▔│░│▔▔│▔▔│▔▔│░│▔▔│▔▔│▔▔│░│▔▔│▔▔│▔▔│  │
└─────────────────────────────────────────────────────────┘
```

---

### Variant 10: Numeric Only

**Инфо:** Без визуального bar. Только цифры: Entry, Current, Low, High, Range%, Volatility.

**Плюсы:** Максимально компактный
**Минусы:** Нет визуальной информации

```
┌─────────────────────────────────────────────────────────┐
│ ENTRY: 8.0995  │  CURRENT: 8.44 (+42%)               │
│ LOW: 7.77 (-4%) │  HIGH: 9.78 (+21%)                 │
│ RANGE: 25%      │  VOLATILITY: 2.01                  │
└─────────────────────────────────────────────────────────┘
```

---

### Current (исходный)

**Инфо:** Текущий вариант с bar и маркерами ENTRY/CURRENT.

```
┌─────────────────────────────────────────────────────────┐
│ [███████████████████████████████]                         │
│      │ENTRY│              │CURRENT│                  │
└─────────────────────────────────────────────────────────┘
```

## Паттерн секции варианта

```html
<div class="variant-section">
    <h2>Variant N: Название варианта</h2>
    <div class="info-box">
        <b>Инфо:</b> Описание варианта
        <br><b>Плюсы:</b> Преимущества
        <br><b>Минусы:</b> Недостатки
    </div>
    <div class="variant-box">
        <div class="variant-N-bar">
            <!-- Контент bar -->
        </div>
        <div class="price-labels-inline">
            <span class="low">7.77 (Low)</span>
            <span class="high">9.78 (High)</span>
        </div>
        <div class="meta-info">
            <span>Метрика 1: <strong>значение</strong></span>
            <span>Метрика 2: <span class="positive">+значение</span></span>
        </div>
    </div>
</div>
```

---

## CSS классы

| Класс | Назначение |
|-------|------------|
| `.variant-section` | Секция варианта |
| `.variant-box` | Контейнер варианта |
| `.variant-N-bar` | Основной bar |
| `.info-box` | Блок с инфо-подписью |
| `.price-labels-inline` | Ценовые метки |
| `.meta-info` | Доп. информация |
| `.positive` | Зелёный цвет (+) |
| `.negative` | Красный цвет (-) |
