## 7. ВЫВОДЫ И ПРЕДПОЛОЖЕНИЯ

### 🚨 Атака (установлено)

1. **Вектор**: Вредонос `_cache_AccountRestore2.exe` запущен из профиля пользователя `Склад1` (папка `Music\res\res\`)
2. **Метод**: Password spraying + Pass-the-Hash для горизонтального перемещения
3. **Жертвы**: Склад1 → Зал1-8, Директор, Менеджер, Администратор
4. **Результат**: Злоумышленник получил **локального администратора** на SERVER2

### ❓ Неизвестно (требует дополнительного расследования)

1. **Как попал `_cache_AccountRestore2.exe` на машину?** — Фишинг? USB? Уязвимость?
2. **Что делал вредонос после успешного входа?** — Загрузка дополнительной нагрузки?
3. **Сетевой артефакт**: адрес `95.181.52.102:3737` в логах — потенциальный C2
4. **Есть ли** аналогичные артефакты на других машинах сети?
